Save As Image Type

## Summary (132 chars max)

## Description
Save As Image Type adds a simple right-click menu to convert and save any image on the web in your preferred format.

**Supported output formats:**
• PNG — Lossless with transparency
• JPEG — Compressed, great for photos
• WebP — Modern format, small file sizes
• BMP — Uncompressed bitmap
• GIF — Classic web format

**Key features:**
• Works with ALL source image types — WebP, AVIF, SVG, PNG, JPEG, GIF, BMP, TIFF, and more
• 100% local processing — no images are uploaded anywhere
• Preserves the original filename
• Handles transparency correctly (white background for JPEG)
• Lightweight — no popups, no settings pages, no bloat
• Zero data collection — see our privacy policy

**How to use:**
1. Right-click any image on a webpage
2. Select "Save Image As…" from the menu
3. Choose your target format (PNG, JPEG, WebP, BMP, or GIF)
4. Save the converted image

Perfect for designers, developers, and anyone who needs to quickly convert WebP images to PNG or JPEG without leaving the browser.
